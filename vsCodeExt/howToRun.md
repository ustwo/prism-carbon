# VS code extension

# prerequisites
npm (May have to be updated)
yeoman 
esbuild (npm i --save-dev esbuild)


# How to run

inside src/extension.ts
run the file 
This should open a new VS code window
Then press "Ctrl+Shift+P"
Type Hello world
If this has worked a little pop up should show up with the text
    "Hello world from EstimatingCarbon"
